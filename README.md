# GREEN-API Тестовое задание

HTML-страница для работы с GREEN-API WhatsApp (Telegram) методами.

## Функциональность

- Получение настроек инстанса (getSettings)
- Проверка состояния инстанса (getStateInstance)
- Отправка текстовых сообщений (sendMessage)
- Отправка файлов по URL (sendFileByUrl)
- Автоматическая проверка статуса подключения
- Адаптивный дизайн на Bootstrap 5

## Технологии

- HTML5
- CSS3 (Bootstrap 5 + кастомные стили)
- JavaScript (ES6+)
- GREEN-API
